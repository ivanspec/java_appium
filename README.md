# JavaAppiumCucumber
  This repo for POC mobile automation using java cucumber.

## Requirement
- JDK Java 8
- Maven
- IDE Tools (Eclipst, InteliJCE, VSCode) 

## APK Test

And extract this zip in this repo. If link not work, please contact me.

## Running in Maven

```console
mvn clean test
```
